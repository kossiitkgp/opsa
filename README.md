About The Project

<div align="center">
<a href="https://github.com/kossiitkgp/opsa">

This project is a monorepo containing highly confidential tools developed within a unique n-sided polygonal building. It includes components like Digester and Excretor, designed for various critical operations.

Getting Started

To set up a local instance of the application, follow the steps below.

Prerequisites

The following dependencies are required to be installed for the project to function properly:

    Rust
    Bash

# Install Rust by following instructions from the official website
https://www.rust-lang.org/tools/install

Docker and Docker Compose
Bash

# Install Docker and Docker Compose
# Add your user to the docker group
sudo usermod -aG docker $USER

GNU Make
Bash

# Ensure GNU Make is installed

cargo-watch
Bash

cargo install cargo-watch

sqlx-cli
Bash

cargo install sqlx-cli

Node.js and npm
Bash

    # Make sure you have Node.js installed

<p align="right">(<a href="#top">back to top</a>)</p>

Installation

Now that the environment has been set up and configured to properly compile and run the project, the next step is to install and configure the project locally on your system.

    Clone the repository
    Bash

git clone https://github.com/kossiitkgp/opsa.git

Navigate to the project directory
Bash

cd ./opsa

Download a Slack archive file from your required workspace. How-to

Create .env file in the project root from .env.template

Run the Digester
Bash

make digest FILE="/path/to/your/slack/archive.zip"

Run the development environment
Bash

make dev

Install frontend dependencies in the garnisher directory
Bash

cd ./garnisher
npm install

Start the frontend server in the garnisher directory
Bash

    npm run dev
